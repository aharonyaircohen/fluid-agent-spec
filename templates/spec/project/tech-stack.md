# Tech Stack

## Backend

* Payload CMS (Node + Express + TypeScript)
* PostgreSQL on Supabase
* GraphQL (built-in from Payload)
* PayPlus (Redirect + Webhook integration) 

## Frontend

* Next.js (App Router + TypeScript)
* TailwindCSS
* Custom Design Tokens (ocean/aqua/violet + Heebo + Space Grotesk)
* graphql-request

## Monorepo / Dev Tools

* pnpm workspaces
* Turborepo
* ESLint + Prettier
* Supabase (DB only)
* Playwright (later, for UI flow tests)
* AIOS Task Manager integration (task specs + prompt files)
